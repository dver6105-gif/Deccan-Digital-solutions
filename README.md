# Deccan Digital Solutions

Production-ready portfolio and downloadable app hosting site for a freelance developer serving small businesses.

## Stack

- Next.js App Router with TypeScript
- Tailwind CSS and shadcn/ui-style components
- Framer Motion scroll animations
- React Hook Form and Zod contact form
- Vercel Analytics and Speed Insights
- Dynamic sitemap and robots routes

## Run Locally

```bash
npm install
npm run dev
```

## Deploy on Vercel

1. Push this repository to GitHub.
2. Import it in Vercel.
3. Keep the default framework preset as Next.js.
4. Deploy.

The site runs on Vercel's free tier. Download files are currently placed in `public/downloads`; for larger production binaries, replace the links with Vercel Blob or GitHub Release URLs.
